Desenvolvimento Web II

  • Carga horária: 80 horas;
  • Quantidade de aulas: ± 10 aulas por bimestre (conteúdo, prática, trabalhos e provas);

Competência:

Geral:

  • Aplicar conceitos, métodos, técnicas através de ferramentas para o desenvolvimento de sistemas Web utilizando linguagens de programação para demonstração conceitual e aplicada aos contextos de mercado com a utilização de Frameworks em nível avançado. Desenvolver projetos de software com a utilização de Frameworks, garantindo a resolução algorítmica de problemas. Elaborar soluções eficazes, eficientes, efetivas e sustentáveis de sistemas de informação, considerando aspectos tecnológicos, econômicos, sociais e ambientais. Construir software para informatização de sistemas avaliando sua qualidade técnica, testando o seu funcionamento, e validando seu atendimento às necessidades de eficácia, eficiência, efetividade e sustentabilidade desses sistemas.

Específica:

  • Utilizar frameworks para criação, organização e estruturação de páginas web: Desenvolvimento de sistemas dinâmicos / Projeto (design) de interface com usuários / Frameworks modernos: Amber, Vue, React, Angular / Linguagem de template do Angular / Diretivas (estruturais e de atributos) / Data binding (one-way, two-way) Eventos (event-binding)
  • Utilizar o servidor de aplicações e banco de dados para distribuição de aplicações web: Node.JS / MongoDB / Express
  • Utilizar linguagens de programação e frameworks para criar páginas dinâmicas, com validações, acesso a dados e consumo de serviços web externos: Desenvolvimento de Sistemas Dinâmicos / Arquitetura MVC / Linguagens HTML, CSS e JavaScript / Requisições Assíncronas (serviço HttpClient) / Comunicação entre front-end e back-end

Sistema de Avaliação

Avaliação prática (AP):

  • Trabalho - Blog: Pontos: 10,0 | Peso: 2,5 (Grupo)
  • Prova - Teórica: Pontos: 10,0 | Peso: 2,5 (Individual)

Avaliação teórica (AT):

  • Trabalho - Projeto: Pontos: 10,0 | Peso: 5,0 (Grupo)
